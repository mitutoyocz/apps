Stránky aplikační podpory

Tato stránka představuje základní rozcestník (index) a seznam pro několik paralelních témat týkajících se zejména komunálních měřidel a déle systémů pro sběr a zpracování dat.


Měření a zpracování dat (measuring and data processing)

  • Připojení komunálních měřidel Mitutoyo k Android zařízením.
    Connecting of Mitutoyo small instruments with Android devices.
  • Příklad JavaScript řešení pro záznam a grafickou prezentaci dat ve společném grafu (až 3 kanály).
    JavaScript example for recording and graphic data presentation (in common diagram).
  • Příklad JavaScript řešení pro záznam a grafickou prezentaci dat v oddělených grafech (až 3 kanály).
    JavaScript example for recording and graphic data presentation (in separate diagrams).

Automatizace (automation)

  • Video QuickImage prototypová instalace pro automatickou kontrolu. Ukázka schopnosti plně automatizovaného běhu a možností napojení na další zařízení.
    Video: QuickImage prototype for automatic product sorting. Shows the possibility for full automatic run and for independent hardware control.
  • Laser Scan Micrometer with the FANUC robot. Ukázka možnosti spolupráce robota s měřícím zařízením.
    Video: Laser Scan Micrometer with the FANUC robot. Automation example.

Lineární odměřovací pravítka (linear scales)

  • Samostatný web věnovaný NC i DRO lineárním odměřovacím pravítkům.
    Stand-alone website for DRO and NC linear scales. In Czech language only.

O těchto stránkách

Obsah těchto stránek je určen pro poskytování aplikační podpory. Jde o prezentaci podrobnějších informací k některým produktům o demonstraci příkladů úspěšných řešení.
Stránky poskytují doplňující infomace k hlavním stránkám společnosti www.mitutoyo.cz.
